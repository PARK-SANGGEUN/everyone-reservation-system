접속 메뉴

교사는 상담 가능 시간을 등록하고 예약 현황을 확인합니다.
학생·학부모는 교사가 제공한 링크로만 예약 가능합니다.

교사용 관리 화면 학생·학부모 예약 화면(테스트)

※ 실제 예약은 반드시 교사가 배부한 링크(apply.html?teacher=TCH-XXXX)로 진행하세요.

사용 안내

교사용
  • 구글 로그인 → 교사코드 자동 생성
  • 월 달력에서 날짜 선택 → 시간 슬롯 열기
  • 예약 현황 실시간 반영 / 엑셀 다운로드
학생용
  • 교사가 준 링크로 접속
  • 가능한 날짜/시간 선택
  • 1인 1회 예약(취소 후 재예약 가능)
학부모용
  • 교사가 준 링크로 접속
  • 자녀 이름/연락처 입력
  • 예약·취소 즉시 반영
